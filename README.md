# Government Data: Yield Curve Analysis

This project analyzes U.S. Treasury yield curves using Python and FRED data.  
It reproduces key concepts from Exercises 3 and 4: bull and bear steepening,  
and the relationship between the 10Y–2Y spread and the S&P 500 index.

## Features

- Visualization of Treasury yield curve dynamics
- Analysis of bull and bear steepening periods
- Double-axis plot: 10Y–2Y spread vs S&P 500
- Analysis of economic conditions and Federal Reserve responses

## Instructions

1. Open `yield_curve_analysis.ipynb` in Google Colab.
2. Replace `YOUR_FRED_API_KEY` with your personal FRED API key.
3. Run all cells to reproduce the plots and analysis.

## Key Takeaways

- Yield curve slope reflects expectations of growth and monetary policy.
- Bull steepening occurs in downturns; bear steepening occurs in expansions.
- The 10Y–2Y spread is a strong forward-looking recession indicator.
- Comparing the spread to the S&P 500 highlights market reactions to economic changes.
